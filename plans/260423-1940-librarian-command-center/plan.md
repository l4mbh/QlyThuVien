## Overview
Nâng cấp hệ thống báo cáo thành trung tâm điều hành dành cho thủ thư, tập trung vào 2 trục: Vận hành (Operational) và Quản lý Kho (Collection). Thay thế các biểu đồ chung chung bằng các danh sách hành động thực tế (Actionable Lists) và số liệu đối soát tài chính/tài sản.

## Tech Stack
- Frontend: React + Vite, TanStack Table (Standard), shadcn/ui.
- Backend: Node.js + Express, Prisma (Aggregations).
- PDF/Print: jsPDF, CSS @media print.
