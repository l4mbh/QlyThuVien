# Plan: Dynamic Breadcrumb Component
Created: 2026-04-24
Status: 🟡 In Progress

## Overview
Implement a reusable, dynamic breadcrumb component for the Library Management System. The component will automatically generate navigation links based on the current route hierarchy, using React Router v7's `useMatches` hook and route `handle` metadata.

## Tech Stack
- **Framework**: React 19 (Vite)
- **UI Library**: shadcn/ui (Breadcrumb component)
- **Routing**: React Router v7
- **Icons**: Lucide React
