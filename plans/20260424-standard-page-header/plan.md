# Plan: Standard Page Header Component
Created: 2026-04-24
Status: 🟡 In Progress

## Overview
Implement a unified `PageHeader` component that encapsulates breadcrumbs, page titles, descriptions, and action buttons. This component will leverage route metadata for titles and descriptions to maintain DRY principles while providing flexibility for page-specific actions.

## Tech Stack
- **Framework**: React 19
- **UI Library**: shadcn/ui + Tailwind CSS
- **Routing**: React Router v7 (matches & handle)
